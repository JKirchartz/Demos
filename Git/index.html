Look Smarter (Or Break Everything) With Git

A short guide to git tricks and worthwhile aliases

Forgot to add a file?

Add to last commit


							$>git add path/to/file.ext
							$>git commit --amend

Note: if you've already pushed your previous commit to remote, you may cause headaches for other developers missing your forgotten files

Add all changes to last commit, Keep commit message

$>git commit --all --ammend --no-edit

`--all` adds all changed files

`--no-edit` skips editing the commit message

Save this as an alias like so: `$> git --global alias.caa = 'commit --all --ammend --no-edit'`

Add the wrong file?

Unstage Changes

$>git reset HEAD --

Save this as an alias like so: `$> git --global alias.unstage = 'reset HEAD --'`

Undo Last commit

$>git reset HEAD~1

Save this as an alias like so: `$> git --global alias.undo = 'reset HEAD~1'`

Git Rebase

$>git rebase -i

Notes: Do not rebase commits you've already pushed to remote, you will give everyone headaches

`-i` enables interactive mode

In interactive mode you can re-order commits by changing the order of the lines in the commit message
